Edit distance

Conta il numero minimo di operazioni (inserzioni, cancellazioni, sostituzioni) necessarie per trasformare un testo nell'altro.

Come si calcola: L'algoritmo più comune è la distanza di Levenshtein, che conta le operazioni di modifica tra due stringhe.

Come si ottimizza: Un valore basso indica che i testi sono simili, mentre un valore alto segnala grandi differenze. Per ridurre la distanza, si possono mantenere più parole invariate o minimizzare le modifiche strutturali.
